# Cine-Flix

A portable web application for streaming movies, TV shows & anime.

## Owner
Varun Bala Surya Kondreddi (varunbala2237)

## Repository
[https://github.com/varunbala2237/cine-flix](https://github.com/varunbala2237/cine-flix)

## Features
- Stream movies, TV shows, and anime
- Save search & watch history
- Sub/Dub toggle for anime
- Recommended media section

## Installation
```bash
git clone https://github.com/varunbala2237/cine-flix.git
cd cine-flix
npm install
npm run dev
