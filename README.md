# Cine-Flix

A portable web application for streaming movies, TV shows & anime.

## Features
- Latest & Trending
- Search
- Dynamic Mapping
- Recommendations

## Utilities

This project contains dynamic mapping of TMDb to AniList data.

Source files located at:
- client/src/api/fetchAnimeMedia.js
- client/src/utils/animeUtils.js

## Setup

Create .env file with the following keys:

```env
VITE_TMDB_API_KEY=...
VITE_SOURCE_BASE=...
```

Now, in your terminal type these commands:

```bash
git clone https://github.com/varunbala2237/cine-flix.git
cd cine-flix/client
npm install
npm run dev
```
